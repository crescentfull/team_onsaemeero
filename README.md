## 팀명 : UMODA - 우모다(우리모두다함께) 팀
- 팀원들 각자의 기술에 익숙해지는 것을 목표로 하여, 페이지 단위로 개발.
- 기획 의도 : </br>
 어떤 사이트를 만들지에 앞서, 큰 키워드 4가지(코로나 시대의 비대면,  소비층의 중요대상인 mz세대(20~30대), 우리 한국의 전통문화, 편안함)를 먼저 정하였습니다. 
이것을 토대로 주요 소비자층인 mz세대는 최신트렌드와, 이색적인 경험을 추구하고 또한 점차 한국 전통문화에 대한 관심이 증가함으로써 기존 숙박장소를 고급 호텔보다 감성적인
한옥으로 정하는 추세 변화가 있었습니다. 코로나가 안정화된 후, 포스트 코로나 시대에는 홈술, 홈파티와 같은 사적인 공간에서의 놀이가 더욱 유행할 것이고, 국내여행객은 대폭 증가 할 것으로 생각됩니다. 
이러한 논의 과정을 바탕으로 저희는 주요 소비자 층을 국내 여행을 하는 mz세대라고 정하고 이색적 체험 과 우리 전통문화, 편안하고 사적인 공간, 모든 것을 해결 할 수 있는 한옥 독채 펜션 사이트를 개발하기로 하였습니다.
