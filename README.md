## 💻 구현 기능
### BACKEND
#### 송영록

- 상품상세페이지 후기 및 댓글 등록 기능 구현
- 메인페이지 검색기능 구현

#### 

- offset과 limit을 이용한 페이징기법으로 상품 목록 조회 API
- 최신순, 가격높은순, 가격낮은순, 이름순 정렬을 이용한 상품 목록 조회 API
- 특정 상품 클릭 시, 상품 상세정보 보여주는 상세정보 API

#### 송영록

- 회원가입 API
- jwt와 bcrpyt를 이용한 로그인 API
- 장바구니 상품 추가, 수정, 삭제 API
